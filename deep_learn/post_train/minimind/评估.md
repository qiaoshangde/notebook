如果你想知道你的模型和 Qwen-0.5B、TinyLlama 这些知名小模型相比处于什么水平，可以使用 `lm-evaluation-harness` 框架进行标准化测试。

**步骤：**

1. **安装测试框架**：
    
    Bash
    
    ```
    pip install lm_eval
    ```
    
2. **运行评测命令**： （以 C-Eval 中文榜单为例，这通常全是选择题）
    
    Bash
    
    ```
    # 这里的 --model_args 需要指向你转换成 huggingface 格式的模型路径
    lm_eval --model hf \
        --model_args pretrained=./out/full_sft_512,trust_remote_code=True \
        --tasks ceval-valid \
        --batch_size 4 \
        --device cuda:0
    ```
    
3. **看结果**： 最后会输出一个表格，重点看 `acc` (Accuracy) 分数。
    
    - _预期管理_：对于 <1B 的小模型，C-Eval 分数通常在 25-30 分左右（也就是接近瞎猜的水平），这很正常。重点是看**你的模型是否比预训练(Pretrain)版本的分数高**，这代表 SFT 是否激发了能力。

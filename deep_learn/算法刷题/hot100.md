# 1、字母异位词分组
![[Pasted image 20260131144041.png]]


用到了字典哈希。
先遍历对所有单词进行排序，然后以排序后的词作为键，值是一个列表，然后判断字典中是否有这个键，然后将其添加到这个排序后的键对应的值（列表）中。这样所有的异位词组都会存到这个键对应的列表中。只需要取值，然后进行list就可以了。

代码
```python
class Solution:

    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        mp = collections.defaultdict(list)
        for st in strs:

            key = "".join(sorted(st))

            mp[key].append(st)

        return list(mp.values())
```

在代码中用到了一个叫collections的库，这个库可以帮助我少点检查是否有这个键的操作。直接向里面添加，如果有就可以对这个键对应的值进行操作，如果没有，会新建一个这个键，直接进行操作。

defaultdict（list）这个括号中填写一个数据类型。一旦检测到没有这个键，新建的这个键对应的值的数据类型就是这个默认的数据类型。
soted 就是进行排序。不管扔给他的是字符串还是列表，其返回都是一个列表。

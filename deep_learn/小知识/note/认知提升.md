# 如何去掉自己身上的弱者气息

于所有人，事，物 无所求，一切可接收即可

不是因为讨好形人格，不是因为善良，不是因为脸皮薄，而是因为你的能力弱。
以自己为主体，全方位的提升自己，不浪费一秒

# 不要惧怕任何事
不要惧怕任何事，允许任何事情发生。
只有你觉得对方会给你带来一些什么的时候，你才会害怕，因为你害怕失去这些好处，会很落魄。改善方法就是：**我得不到这些东西我也会过的好好的。**
把目的模糊化，减轻得失，贪欲就降下来了，恐惧也就消失了，无欲则刚！！无所谓训斥自己的贪欲，并且降低我们自己的期望，会让你活得更加游刃有余。

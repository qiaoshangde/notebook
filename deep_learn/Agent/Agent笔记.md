
- **Agent = 干活的智能体**  
    负责思考、规划、调用工具、完成任务。
- **Agent Framework = 造智能体的零件库**  
    给你工具、接口、循环逻辑，让你能拼出一个 Agent。
- **Agent Harness = 管理智能体的操作系统**  
    负责让 Agent 稳定、不崩、不跑偏、能长期跑。

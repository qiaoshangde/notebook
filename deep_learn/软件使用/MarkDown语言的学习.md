# 链接

1、如何创建文档属性
在标题的正下方的第一行用---来展开笔记属性，可以设置标签，别名，日期等信息。
当不在文档正下方时，会产生一条分割线，在幻灯片演示用可以用到。 ^0847fe

---

2、#+空格是一级标题


3、创建链接（最重要的）就是用两个中括号把链接括起来比如：[[我的第一篇笔记]]。当你需要在这个界面，查看链接内的东西时，只需要用ctrl+鼠标对准这个地方即可。

[[我的第一篇笔记#一级标题]]还可以链接到这个笔记的某个标题
[[我的第一篇笔记|我的]]

[[我的第一篇笔记#^fd3145]]链接到某个笔记的某句话
还可以将链接嵌入进来，只需要在中括号前面加一个！即可比如：
嵌入还可以与上面的链接标题和链接文本一起配合使用

![[我的第一篇笔记]]

还可以添加外部链接，直接使用中括号+小括号即可
显示的文字是中括号中的文字，而链接时（）中的，比如
[百度](http://www.baidu.com)



# 文本编辑

1、一级标题就是#+空格  几个#号就是几级标题
2、==两个等号是高亮==
3、**两个星号是加粗**    __两个下划线也是加粗__     **或者直接ctrl + B**
4、*一个星号是斜体*     也可以用下划线      *或者ctrl + i*
5、~~两个波浪线是斜体~~     快捷键改为了   ctrl +shift +d   是添加删除线。ctrl + d


# 快捷键：
ctrl + p 最常用的打开命令面板
ctrl + o是打开
ctrl + n 新建
ctrl +f 寻找
ctrl +h 替换
ctrl + e 切换预览模式（不允许修改）
ctrl +d 删除一行
ctrl +shift +d 添加删除线
ctrl +L 是代办事项
ctrl + shift +k是插入代码段
ctrl + i  是斜体
ctrl +w 是关闭
ctrl +k是添加外部链接

# 注释



被注释的地方[^1]




[^1]:这是注释1

这就是注释，在阅读模式时，可以看到，每次注释不用在文章的末尾，在后面即可，在预览模式下会自动调整到最后。甚至还可以在原位置添加，预览模式下在最后[^zheshi]
还可以用三个中括号，一个为注释，里面两个为链接，链接到其他笔记
